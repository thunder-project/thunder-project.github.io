thunder.TuningModel¶

class thunder.TuningModel(modelfile, var='s')¶

Base class for loading and fitting tuning models.

Parameters:

modelfile : str, or array

Array of input values or specification of a MAT file containing a variable s with input values

var : str, default = ‘s’

Variable name if loading from a MAT file
