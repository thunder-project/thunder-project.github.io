API Reference¶

Loading¶

ThunderContext(sparkcontext) Wrapper for a SparkContext that provides functionality for loading data.

Data types¶

Series(rdd[, index, dims, dtype]) Distributed collection of 1d array data with axis labels.
Images(rdd[, dims, nimages, dtype]) Distributed collection of images or volumes.
TimeSeries(rdd[, index, dims, dtype]) Distributed collection of time series data.
SpatialSeries(rdd[, index, dims, dtype]) Distributed collection of 1d array data with spatial coordinates.
RowMatrix(rdd[, index, dims, nrows, ncols]) Distributed matrix data.

Decoding¶

MassUnivariateClassifier(paramfile) Base class for performing classification.
GaussNaiveBayesClassifier(paramfile[, cv]) Classifier for Gaussian Naive Bayes classification.
TTestClassifier(paramfile) Classifier for TTest classification.

Factorization¶

PCA([k, svdmethod]) Principal components analysis on a distributed matrix.
ICA(c[, k, svdmethod, maxiter, tol, seed]) Independent component analysis on a distributed matrix.
NMF([k, method, maxiter, tol, h0, w_hist, ...]) Non-negative matrix factorization on a distributed matrix.
SVD([k, method, maxiter, tol]) Singular value decomposiiton on a distributed matrix.

Regression¶

RegressionModel Base class for loading and fitting regression models.
MeanRegressionModel(modelfile[, var]) Regression in the form of simple averaging.
LinearRegressionModel(modelfile[, var]) Ordinary least squares linear regression.
BilinearRegressionModel(modelfile[, var]) Bilinear regression with two design matrices.
TuningModel(modelfile[, var]) Base class for loading and fitting tuning models.
GaussianTuningModel(modelfile[, var]) Gaussian tuning model fitting.
CircularTuningModel(modelfile[, var]) Circular tuning model fitting.

Clustering¶

KMeans(k[, maxIterations]) KMeans clustering algorithm.
KMeansModel(centers) Estimated KMeans model and its parameters.

Visualization¶

Colorize([totype, scale, colors]) Class for turning numerical data into colors.

Utilities¶

export(data, outputdir, outputfile, outputformat) Export data to a variety of local formats.
